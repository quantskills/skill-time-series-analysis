# Anti-Patterns

- Starting from low-level helpers when a main API can answer the question.
- Treating Hurst, ADF, or KPSS as a standalone trading signal.
- Comparing log returns on non-positive spread values without checking the transform.
- Reporting a cointegration p-value without beta, residual evidence, and sample length.
- Omitting caveats when tests disagree or the sample is short.
